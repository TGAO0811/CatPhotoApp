# CatPhotoApp
CatPhotoApp from freeCodeCamp.org to learn HTML.

Things Learnt: 
- HTML Tags:
    - <h1>, <h2>, ...: Heading tags
    - <p>: paragraph tag
    - <a href="">: Hyperlink
    - <img src ="">: Image
    - <ul>, <li>: unordered list
    - <figure>, <figcaption> : figure and caption
- HTML Sections
    - <html>: Basic outside
    - <body>: Body section
    - <main>: main section
    - <section>: sub-section
- <!DOCTYPE html>: At the beginning of HTML file
